# 导数样例

## 题目

求函数

$$
f(x) = x^2 + 3x + 1
$$

的导数。

## 思路

分别对多项式的每一项求导：

$$
\frac{d}{dx}x^2 = 2x,\quad \frac{d}{dx}3x = 3,\quad \frac{d}{dx}1 = 0
$$

## 解答

$$
f'(x) = 2x + 3
$$

::: details 易错点
常数项求导为 0。
:::
